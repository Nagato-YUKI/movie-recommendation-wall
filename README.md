# 影荐墙 - 经典高分电影推荐展示站

> 一个精心设计的静态电影推荐网站，收录 13 部高分佳作，提供搜索、筛选、收藏、点赞、评分、留言等丰富互动功能。

## 在线预览

[点击访问在线演示](https://nagato-yuki.github.io/movie-recommendation-wall/)

## 项目介绍

影荐墙是一个专注于经典高分电影推荐的展示站点。网站采用纯前端技术栈构建，无需后端服务即可运行。通过 localStorage 实现数据持久化，让用户的收藏、点赞、评分和留言在刷新页面后依然保留。

## 功能列表

### 核心展示
- **电影卡片展示**：13 部精选高分电影，以卡片形式呈现，包含海报、评分、导演、年份、简介和经典台词
- **分类筛选**：支持按 9 大分类筛选（科幻、悬疑、动画、剧情、爱情、喜剧、战争、传记、历史）
- **实时搜索**：支持按电影名称、原名、导演或简介内容搜索

### 用户互动
- **收藏功能**：点击心形图标收藏/取消收藏电影，数据持久化到 localStorage
- **点赞功能**：为喜欢的电影点赞，显示点赞数量
- **评分功能**：1-5 星评分，用户评分与系统评分独立展示
- **留言板**：访客可提交留言（昵称+内容），留言按时间倒序排列，数据存储在 localStorage

### 展示与体验
- **Lightbox 图片查看**：点击电影海报可放大查看，支持左右箭头切换和键盘导航（ESC 关闭、左右箭头切换）
- **图片懒加载**：页面滚动到可视区域才加载图片，优化首屏速度，未加载时显示 shimmer 动画
- **滚动进度条**：页面顶部显示滚动进度指示条
- **回到顶部**：平滑滚动回到页面顶部
- **滚动渐入动效**：电影卡片进入视口时触发渐入动画

### 其他
- **响应式布局**：完美适配手机、平板和桌面端
- **简历页面**：包含站长个人简历，可从首页导航跳转
- **分享功能**：一键复制页面链接到剪贴板

## 技术栈

- **HTML5** - 语义化标签结构
- **TailwindCSS** - 实用优先的 CSS 框架
- **原生 JavaScript** - 无框架依赖，模块化代码组织
- **localStorage** - 本地数据持久化
- **IntersectionObserver** - 懒加载与滚动动画
- **Google Fonts** - Noto Sans SC 中文字体

## 本地运行

本项目为纯静态网站，无需构建工具，直接用浏览器打开 `index.html` 即可。

或使用本地服务器：

```bash
# Python 3
python -m http.server 8080

# Node.js
npx serve .
```

然后访问 `http://localhost:8080`

## 海报来源

本项目使用的电影海报图片均来自 [Unsplash](https://unsplash.com) 免费图库，遵循 Unsplash License 许可协议，可自由用于个人和商业项目。

由于以下原因，未直接从小宝影院等影视网站爬取海报：
1. **反爬虫机制**：影视网站通常有反爬虫保护，会检测并阻止非浏览器请求
2. **动态加载**：海报图片通过 JavaScript 动态加载，直接 HTTP 请求无法获取真实图片 URL
3. **防盗链**：图片服务器会检查 Referer 头，拒绝来自非本站的直接请求
4. **版权风险**：未经授权爬取受版权保护的海报可能涉及侵权

因此选择使用 Unsplash 的免费高质量图片作为替代方案。
